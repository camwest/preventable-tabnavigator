h2. preventable-tabnavigator

h3. DESCRIPTION:

A Flex 3 component that prevents the default tab switching in the standard TabNavigator and instead dispatches a PreventableTabNavigatorEvent.CHANGING event to notify the user the TabNavigator wants to change tabs.

Useful in using controllers to handle the switching of tabs.

h3. INSTALLATION:

git submodule add git@github.com:camwest/preventable-tabnavigator.git app/flex/com/bigbangtechnology/preventableTabNavigator
